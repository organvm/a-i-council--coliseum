## 2024-05-23 - [Skip to Content Link]
**Learning:** Adding a 'Skip to Content' link is a critical accessibility pattern often overlooked in Next.js App Router setups. By adding a hidden anchor in the root layout and a target ID on the main content area, we ensure keyboard users can bypass navigation efficiently.
**Action:** Always verify the presence of a 'Skip to Content' link in the root layout and ensure the target ID exists on the main content container.
